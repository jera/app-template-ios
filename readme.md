# APP Template iOS
---------------

Esse é o template que contém as seguintes funcionalidades:

 - Login com email e senha
 - Login com Facebook
 - Login com Google
 - Esqueci minha senha
 - Cadastro de usuário com: nome, email, telefone, cpf e senha

Ele é escrito na arquitetura VIPER
